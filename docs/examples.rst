Examples
========

The toolkit includes several demonstration scripts in the ``examples/`` directory.

Interplanetary Transfers
-------------------------

* **Mars Hohmann Transfer**: Classic Earth-to-Mars transfer using patched conics and Lambert's problem.
  Run with: ``python examples/mars_transfer_hohmann_demo.py``

* **Earth Escape**: Calculating departure Hyperbola and C3.
  Run with: ``python examples/earth_escape_demo.py``

Optimization & Targeting
------------------------

* **Gravity Assist Optimized**: Iterative B-plane targeting at the Moon.
  Run with: ``python examples/gravity_assist_optimized_demo.py``

* **N-Body Correction**: Differential correction for high-fidelity paths.
  Run with: ``python examples/nbody_demo.py``
